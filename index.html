Analyze — Result

This repository contains a small data-processing script (execute.py) and source data (data.csv). The GitHub Actions CI will run ruff, execute the script to generate result.json, and publish result.json to GitHub Pages. result.json is NOT committed to the repo and is generated during CI.

Viewing the generated result

After CI runs, result.json is published to GitHub Pages at /result.json. You can view it at the repository Pages URL (for example):

  • Relative URL: /result.json

If your Pages site is served from https://<owner>.github.io/<repo>/ then the file will be available at https://<owner>.github.io/<repo>/result.json.

How to run locally

Requirements: Python 3.11+, pip

  1. Install dependencies: pip install pandas==2.3.0
  2. Run: python execute.py > result.json
  3. Open result.json with your editor or serve it with a static server.

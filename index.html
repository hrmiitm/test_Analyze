This repository contains a small data-processing script (execute.py) that reads data.csv and writes result.json.

How this works

  1. CI (GitHub Actions) runs ruff, then runs python execute.py > result.json.
  2. The workflow uploads the generated result.json as the Pages artifact and then deploys it to GitHub Pages.

Notes:

  • Do not commit result.json — it is produced by CI and published to Pages.

View latest published result.json

If the repository's GitHub Pages deployment is enabled for the workflow, the generated result.json will be available at the published Pages root. Example URL (replace USERNAME and REPO):

https://USERNAME.github.io/REPO/result.json

Locally you can run:

python -m pip install pandas==2.3.0
python execute.py > result.json
